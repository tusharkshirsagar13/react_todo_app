# React + TypeScript Todo App

A fast and responsive Todo application built to demonstrate modern React development practices using TypeScript and Vite.

## ‚ú® Features

* **Tech Stack:** Built with React (Hooks), TypeScript, and Vite for blazing fast performance.
* **State Management:** Uses React Context API for clean and efficient global state management.
* **Persistent Data:** Tasks are saved to the browser's LocalStorage, so your data is safe even after a refresh.
* **Filtering:** specialized routing allows you to easily filter tasks by "All", "Active", and "Completed" statuses.
* **CRUD Operations:** Add, Read, Update (mark as complete), and Delete tasks.
* **Styling:** Clean and responsive UI using standard CSS.
